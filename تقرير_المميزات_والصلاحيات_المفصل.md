# 📊 تقرير شامل عن مميزات النظام والصلاحيات
## نظام إدارة العقارات - Aqar Management System

---

## 🎯 نظرة عامة

**نظام إدارة العقارات** هو منصة متطورة وشاملة تم تطويرها باستخدام أحدث التقنيات (Laravel 11) لتلبية احتياجات قطاع العقارات في المملكة العربية السعودية. 

### 🚀 ما يميز هذا النظام:

✅ **نظام أدوار متقدم** - إدارة دقيقة للصلاحيات  
✅ **نظام مالي متطور** - محاسبة شاملة مع تقارير تفاعلية  
✅ **دعم متعدد اللغات** - العربية والإنجليزية  
✅ **API متكامل** - للتطبيقات المحمولة والتكامل الخارجي  
✅ **واجهة حديثة** - سهلة الاستخدام ومتجاوبة  

### 👥 الفئات المستهدفة:
- **مؤسسات عقارية** - لإدارة أعمالها وموظفيها
- **مطوري العقارات** - لعرض مشاريعهم وإدارة مبيعاتهم  
- **العملاء** - للبحث عن العقارات المناسبة وحجزها
- **المديرين** - لمراقبة الأداء واتخاذ القرارات

---

## 🎭 نظام الأدوار والصلاحيات المتقدم

النظام يوفر **ثلاثة أدوار رئيسية** مع صلاحيات محددة لكل دور، مما يضمن الأمان والتنظيم المثالي.

---

### 👑 1. مدير النظام (Admin)

**🔑 الصلاحيات الكاملة:**
- **إدارة شاملة** لجميع جوانب النظام
- **إدارة المستخدمين** - إنشاء، تعديل، حذف، تصدير البيانات
- **إدارة المنشآت** - التحكم في الحالة والتحقق والمميز
- **إدارة المنتجات** - مع إدارة المعارض والتعليقات
- **إدارة العقود** - مع النظام المالي المتكامل
- **إدارة الأدوار والصلاحيات** - تخصيص الصلاحيات
- **التقارير والإحصائيات** - تقارير شاملة مع إمكانية التصدير
- **إعدادات النظام** - تخصيص النظام العام

**🛠️ الوظائف المتاحة:**
- لوحة تحكم تفاعلية مع إحصائيات مباشرة
- إدارة المستخدمين مع إمكانية البحث والفلترة
- إدارة المنشآت مع التحكم في الترخيص والتحقق
- إدارة المنتجات مع معرض الصور والفيديو
- إدارة الحجوزات مع تقويم تفاعلي
- إدارة العقود مع النظام المالي المتكامل
- إدارة الأدوار والصلاحيات المفصلة
- التقارير المالية الشاملة مع الرسوم البيانية

**📊 الصلاحيات التقنية:**
- عرض وإنشاء وتعديل وحذف المستخدمين
- عرض وإنشاء وتعديل وحذف المنشآت  
- عرض وإنشاء وتعديل وحذف المنتجات
- عرض وإنشاء وتعديل وحذف الحجوزات
- عرض وإنشاء وتعديل وحذف العقود
- إدارة الأدوار والصلاحيات
- الوصول لإعدادات النظام
- عرض وتصدير جميع التقارير

### 🏢 2. المنشأة العقارية (Facility)

**🎯 الصلاحيات المتخصصة:**

- **إدارة الحجوزات** - حجوزات المنشأة مع تقويم تفاعلي

- **إدارة العروض والعقود** - إنشاء وإدارة العروض والعقود
- **التقارير المالية** - تقارير مفصلة لأداء المنشأة

**🛠️ الوظائف المتاحة:**
- **لوحة تحكم متقدمة** - إحصائيات شاملة مع الرسوم البيانية
- **إدارة المنتجات** - مع معارض الصور والفيديو والخصائص
- **إدارة الحجوزات** - مع تقويم تفاعلي ونظام المواعيد
- **النظام المالي** - محاسبة متكاملة مع القيود التلقائية
- **إدارة العروض** - عروض البيع والإيجار مع تتبع الأداء
- **إدارة العقود** - عقود شاملة مع خطة الدفع والعمولات

**🎨 الميزات المتقدمة:**
- **تخصيص التصميم** - ألوان مخصصة وخطوط وتخطيط
- **معرض الوسائط** - صور وفيديو عالي الجودة
- **نظام التقييمات** - تقييمات العملاء ومراجعاتهم
- **تقويم المواعيد** - جدولة الحجوزات والزيارات
- **النظام المالي** - قيود محاسبية تلقائية وتقارير مالية

## 🏢 مميزات إدارة المنشآت العقارية

النظام يوفر **أدوات متطورة** لإدارة المنشآت العقارية مع تخصيص كامل للهوية البصرية والتسويقية.

---

### 📋 1. إدارة معلومات المنشأة

**🏢 المعلومات الأساسية:**
- **اسم المنشأة** ووصف شامل
- **العنوان الكامل** مع الموقع الجغرافي الدقيق
- **بيانات الاتصال** - الهاتف والبريد الإلكتروني والموقع الإلكتروني
- **رقم الترخيص** وتاريخ انتهائه
- **الموقع على الخرائط** - GPS مع رابط خرائط جوجل

**⭐ المعلومات المتقدمة:**
- **الشعار وصورة الغلاف** - صور عالية الجودة
- **التصنيف والمدينة** - تصنيف مناسب للمنشأة
- **نظام التقييمات** - تقييمات العملاء وعدد التقييمات
- **حالة النشاط** - نشط/غير نشط، محقق/غير محقق، مميز

---

**🌈 نظام الألوان المتقدم:**
- **الألوان الأساسية** - اللون الرئيسي والثانوي والتأكيدي
- **ألوان النصوص** - لون النص الأساسي والثانوي
- **لون الخلفية** - تخصيص لون الخلفية العامة

**🎭 خيارات التصميم:**
- **عائلة الخطوط** - خطوط عربية وإنجليزية متنوعة
- **نمط التخطيط** - تخطيطات مختلفة للصفحات
- **نمط الأزرار** - أشكال مختلفة للأزرار
- **موضع الشعار** - موقع الشعار في الصفحة
- **التأثيرات** - تفعيل/إلغاء الرسوم المتحركة والتأثيرات

**🖼️ الخلفية الرئيسية:**
- **نوع الخلفية** - لون/صورة/تدرج
- **قيمة الخلفية** - الصورة أو اللون المحدد
- **شفافية الطبقة** - شفافية الطبقة العلوية

**💻 التخصيص التقني:**
- **CSS مخصص** - إضافة أكواد CSS خاصة
- **وسائل التواصل الاجتماعي** - روابط الحسابات
- **SEO** - كلمات مفتاحية ووصف للمحركات

---

### 🏠 3. إدارة المنتجات العقارية

**📝 المعلومات الأساسية:**
- **العنوان والوصف** - معلومات مفصلة بالعربية والإنجليزية
- **الموقع الجغرافي** - عنوان دقيق مع GPS
- **معرض الصور** - صور عالية الجودة للعقار
- **فيديو توضيحي** - جولة في العقار
- **الفئة والحالة** - تصنيف العقار وحالته

**🔧 الخصائص التقنية:**

- **المساحة الإجمالية** - المساحة بالمتر المربع
- **رقم الطابق** وعدد الطوابق الكلية
- **أماكن الوقوف** - عدد أماكن السيارات
- **التأثيث** - مفروش/غير مفروش

**💰 المعلومات المالية:**
- **السعر الأساسي** - سعر العقار
- **العروض المتاحة** - عروض البيع والإيجار
- **تواريخ التوفر** - متى يصبح العقار متاحاً
- **معلومات الاتصال** - للاستفسارات
