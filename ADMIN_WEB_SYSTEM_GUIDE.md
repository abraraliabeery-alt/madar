# دليل النظام الويب للأدمن - النظام المالي المتكامل

## نظرة عامة

تم تطوير نظام ويب متكامل للأدمن لإدارة النظام المالي الجديد بواجهة حديثة وتفاعلية، مع تصميم متجاوب وميزات متقدمة.

## المكونات المطورة

### 1. **الكونترولر الرئيسي**
```
app/Http/Controllers/Admin/AdminFinancialController.php
```

#### الوظائف الرئيسية:
- **لوحة المعلومات**: `dashboard()` - إحصائيات شاملة ورسوم بيانية
- **إدارة العروض**: `offers()` - عرض وإدارة جميع العروض
- **إدارة العقود**: `contracts()` و `contractDetails()` - إدارة العقود وتفاصيلها
- **إدارة المدفوعات**: `payments()` - مراجعة وتأكيد المدفوعات
- **التقارير الشاملة**: `reports()` - تقارير مالية متقدمة
- **القيود المحاسبية**: `accountingEntries()` - عرض وإدارة القيود

### 2. **صفحات الويب (Views)**

#### أ. Layout الأساسي
```
resources/views/admin/financial/layout.blade.php
```
**المميزات:**
- تصميم RTL متجاوب
- شريط تنقل متطور
- شريط جانبي تفاعلي
- أنظمة إشعارات متقدمة
- تحميل تلقائي للمكتبات (Bootstrap 5 RTL, Font Awesome, Chart.js)

#### ب. لوحة المعلومات
```
resources/views/admin/financial/dashboard.blade.php
```
**العناصر:**
- **إحصائيات رئيسية**: إجمالي الإيرادات، العمولات، العقود، العروض
- **تنبيهات ذكية**: العقود المعلقة، الفواتير المتأخرة، المدفوعات المعلقة
- **رسوم بيانية**: إيرادات شهرية تفاعلية
- **أفضل المؤسسات**: ترتيب حسب الأداء
- **الأنشطة الحديثة**: أحدث العقود والمدفوعات

#### ج. إدارة العروض
```
resources/views/admin/financial/offers.blade.php
```
**الميزات:**
- **فلترة متقدمة**: حسب المؤسسة، النوع، الحالة
- **بحث ذكي**: في أسماء المنتجات والعناوين
- **إحصائيات فورية**: عدد العروض، القيمة الإجمالية
- **إجراءات متعددة**: تفعيل/إلغاء، تصدير
- **عرض تفاصيل**: مودالات تفاعلية للتفاصيل

#### د. إدارة العقود
```
resources/views/admin/financial/contracts.blade.php
resources/views/admin/financial/contract-details.blade.php
```
**الوظائف:**
- **إدارة شاملة**: عرض، تعديل، تحديث الحالة
- **تفاصيل متكاملة**: معلومات الأطراف، الفواتير، المدفوعات
- **تتبع المدفوعات**: شريط تقدم، نسبة السداد
- **القيود المحاسبية**: عرض جميع القيود المرتبطة
- **تحميل وطباعة**: تصدير العقود بصيغ متعددة

### 3. **نظام التوجيه (Routes)**

تم إضافة مجموعة شاملة من الطرق في:
```
routes/admin.php
```

#### الطرق الرئيسية:
```php
// لوحة المعلومات المالية
Route::get('/financial/dashboard', [AdminFinancialController::class, 'dashboard']);

// إدارة العروض
Route::get('/financial/offers', [AdminFinancialController::class, 'offers']);

// إدارة العقود
Route::get('/financial/contracts', [AdminFinancialController::class, 'contracts']);
Route::get('/financial/contracts/{id}', [AdminFinancialController::class, 'contractDetails']);
Route::put('/financial/contracts/{id}/status', [AdminFinancialController::class, 'updateContractStatus']);

// إدارة المدفوعات
Route::get('/financial/payments', [AdminFinancialController::class, 'payments']);
Route::post('/financial/payments/{id}/confirm', [AdminFinancialController::class, 'confirmPayment']);

// التقارير والقيود المحاسبية
Route::get('/financial/reports', [AdminFinancialController::class, 'reports']);
Route::get('/financial/accounting-entries', [AdminFinancialController::class, 'accountingEntries']);
```

### 4. **ملفات JavaScript والتفاعل**

```
public/js/admin-financial.js
```

#### الوظائف المتقدمة:
- **فلترة تلقائية**: تطبيق الفلاتر أثناء الكتابة
- **تحديد متعدد**: إدارة العناصر المحددة
- **تصدير ذكي**: تصدير بصيغ متعددة (Excel, PDF, CSV)
- **مودالات ديناميكية**: تحميل المحتوى تلقائياً
- **إشعارات Toast**: نظام إشعارات حديث
- **اختصارات لوحة المفاتيح**: Ctrl+E للتصدير، Ctrl+A للتحديد
- **تحديث تلقائي**: تحديث الإحصائيات كل 5 دقائق

#### أمثلة الاستخدام:
```javascript
// عرض إشعار
showNotification('تم حفظ البيانات بنجاح', 'success');

// تنسيق العملة
formatCurrency(150000); // "150,000.00 ر.س"

// تأكيد حذف
if (confirmDelete('هل تريد حذف هذا العقد؟')) {
    // تنفيذ الحذف
}
```

### 5. **تصميم CSS المتقدم**

```

```

#### المميزات التصميمية:
- **متغيرات CSS**: نظام ألوان موحد
- **تدرجات لونية**: تصميم عصري بالتدرجات
- **رسوم متحركة**: انتقالات سلسة
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **وضع الطباعة**: تحسينات خاصة للطباعة

## الميزات الرئيسية

- 📊 **إحصائيات حية**: تحديث تلقائي كل 5 دقائق
- 📈 **رسوم بيانية**: Chart.js للإيرادات الشهرية
- 🚨 **تنبيهات ذكية**: العقود المعلقة والفواتير المتأخرة
- 🏆 **ترتيب المؤسسات**: أفضل أداء حسب العقود

### 2. **إدارة متقدمة للعروض**
- 🔍 **بحث وفلترة**: متعددة المعايير
- 📊 **إحصائيات فورية**: عدد العروض والقيمة الإجمالية
- ⚡ **إجراءات سريعة**: تفعيل/إلغاء مجمع
- 📥 **تصدير متعدد**: Excel, PDF, CSV

### 3. **إدارة شاملة للعقود**
- 📋 **عرض تفصيلي**: جميع معلومات العقد
- 👥 **معلومات الأطراف**: العميل، المالك، العقار
- 💰 **تتبع المدفوعات**: شريط تقدم ونسبة السداد
- 🧾 **الفواتير**: عرض وإدارة الفواتير
- 📊 **القيود المحاسبية**: جميع القيود المرتبطة

### 4. **نظام مدفوعات متطور**
- ✅ **تأكيد/رفض**: مراجعة المدفوعات المعلقة
- 🔍 **فلترة متقدمة**: حسب الحالة، الطريقة، التاريخ
- 📊 **إحصائيات**: إجمالي المدفوعات وتوزيعها
- 🔔 **إشعارات**: تنبيهات للمدفوعات المعلقة

### 5. **تقارير مالية شاملة**
- 📈 **تقرير العملاء**: أداء العملاء ومعدل السداد
- 🏢 **تقرير الملاك**: العمولات وصافي المستحقات
- 🏬 **تقرير المؤسسات**: أداء كل مؤسسة
- 🧮 **القيود المحاسبية**: تفصيل كامل للقيود

## استكشاف الأخطاء

### 1. **مشاكل شائعة**

#### خطأ في تحميل CSS/JS:
```bash

```

```bash
chmod -R 755 storage/
chmod -R 755 bootstrap/cache/
```

#### خطأ في قاعدة البيانات:
```bash
php artisan migrate
php artisan db:seed
```
