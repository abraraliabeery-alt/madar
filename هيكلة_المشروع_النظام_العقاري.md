# هيكلة المشروع - نظام إدارة العقارات

## 📁 هيكل المشروع العام

```
aqar/
├── app/
│   ├── Console/Commands/          # أوامر Artisan المخصصة
│   ├── Exceptions/               # معالجة الأخطاء
│   ├── Exports/                  # تصدير البيانات (Excel, PDF)
│   ├── Helpers/                  # المساعدات والدوال المساعدة
│   ├── Http/
│   │   ├── Controllers/          # المتحكمات
│   │   │   ├── Admin/           # متحكمات الأدمن
│   │   │   ├── Api/             # متحكمات API
│   │   │   ├── Auth/            # متحكمات المصادقة
│   │   │   ├── Client/          # متحكمات العميل
│   │   │   ├── Facility/        # متحكمات المنشأة
│   │   │   └── Public/          # متحكمات الموقع العام
│   │   └── Middleware/          # الوسطاء (Middleware)
│   ├── Models/                   # النماذج (Models)
│   ├── Notifications/           # الإشعارات
│   ├── Policies/                # السياسات الأمنية
│   ├── Providers/               # مقدمي الخدمات
│   └── Services/                # خدمات الأعمال
├── database/
│   ├── factories/               # مصانع البيانات الوهمية
│   ├── migrations/              # هجرات قاعدة البيانات
│   └── seeders/                 # بذور البيانات
├── resources/
│   ├── lang/                    # ملفات الترجمة
│   └── views/                   # قوالب Blade
├── routes/                      # ملفات التوجيه
└── public/                      # الملفات العامة
```

## 🗄️ قاعدة البيانات - الجداول الرئيسية

### 1. المستخدمين والصلاحيات
- **users** - جدول المستخدمين الأساسي
- **roles** - الأدوار (admin, facility, client)
- **permissions** - الصلاحيات
- **permission_role** - ربط الصلاحيات بالأدوار
- **user_facility_role** - ربط المستخدمين بالمنشآت والأدوار

### 2. المنشآت والعقارات
- **facilities** - المنشآت العقارية
- **facility_categories** - فئات المنشآت
- **products** - المنتجات/العقارات
- **categories** - فئات العقارات
- **features** - مميزات العقارات
- **attributes** - خصائص العقارات

### 3. الحجوزات والعقود
- **bookings** - الحجوزات
- **contracts** - العقود
- **offers** - العروض
- **appointments** - المواعيد

### 4. المالية
- **invoices** - الفواتير
- **payments** - المدفوعات
- **accounting_entries** - القيود المحاسبية
- **chart_of_accounts** - دليل الحسابات

### 5. المحتوى والترجمة
- **pages** - الصفحات الثابتة
- **faqs** - الأسئلة الشائعة
- **notifications** - الإشعارات
- **translations** - جداول الترجمة (user_translations, product_translations, etc.)

## 🎭 الأدوار والصلاحيات

### الأدوار الرئيسية:
1. **Admin (الأدمن)**
   - إدارة شاملة للنظام
   - إدارة المستخدمين والمنشآت
   - إدارة العقود والفواتير
   - التقارير والإحصائيات

2. **Facility (المنشأة)**

   - إدارة الحجوزات والمواعيد

3. **Client (العميل)**
   - تصفح المنتجات والعقارات
   - إدارة الحجوزات والعقود
   - المفضلة والتعليقات
   - الملف الشخصي

- **users.view/create/edit/delete** - إدارة المستخدمين
- **facilities.view/create/edit/delete** - إدارة المنشآت

- **bookings.view/create/edit/delete** - إدارة الحجوزات
- **contracts.view/create/edit/delete** - إدارة العقود
- **roles.view/create/edit/delete** - إدارة الأدوار
- **permissions.view/create/edit/delete** - إدارة الصلاحيات

## 🔄 العلاقات بين النماذج

### User Model:
- belongsToMany(Facility) - المستخدم يمكن أن ينتمي لعدة منشآت
- hasMany(Product) - المستخدم يمكن أن يملك عدة منتجات
- hasMany(Booking) - المستخدم يمكن أن يحجز عدة مرات
- hasMany(Contract) - المستخدم يمكن أن يوقع عدة عقود
- morphedByMany(Product, Facility) - المفضلة

### Facility Model:
- belongsTo(User) - المنشأة تنتمي لمستخدم (المالك)
- hasMany(Product) - المنشأة لديها عدة منتجات
- hasMany(Contract) - المنشأة لديها عدة عقود
- hasManyThrough(Booking, Product) - حجوزات من خلال المنتجات

### Product Model:
- belongsTo(Facility) - المنتج ينتمي لمنشأة
- belongsTo(Category) - المنتج ينتمي لفئة
- hasMany(Booking) - المنتج يمكن حجزه عدة مرات
- hasMany(Offer) - المنتج يمكن أن يكون له عدة عروض

## 📱 API Endpoints

### Authentication:
- POST /api/v1/register - التسجيل
- POST /api/v1/login - تسجيل الدخول
- POST /api/v1/logout - تسجيل الخروج

### Products:
- GET /api/v1/products - قائمة المنتجات
- GET /api/v1/products/{id} - تفاصيل منتج
- GET /api/v1/products/featured - المنتجات المميزة
- GET /api/v1/products/search - البحث في المنتجات

### Facilities:
- GET /api/v1/facilities - قائمة المنشآت
- GET /api/v1/facilities/{id} - تفاصيل منشأة
- GET /api/v1/facilities/{id}/products - منتجات المنشأة

### Bookings:
- GET /api/v1/bookings - حجوزات المستخدم
- POST /api/v1/bookings - إنشاء حجز جديد
- PUT /api/v1/bookings/{id} - تحديث حجز
- DELETE /api/v1/bookings/{id} - إلغاء حجز
